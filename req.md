Project Overview DiteUp.com is a focused single-product e-commerce website with a conversion-optimized landing page, secure online ordering through Razorpay, Meta (Facebook) Pixel for ad tracking, and a complete admin panel with order, payment, user, and inventory management. The platform is built so that the owner can run paid ads on Meta with full conversion tracking and manage the entire business — orders, stock, customers, and emails — from one admin dashboard. Key Highlights • Conversion-optimized single-product landing page with image slides and product videos. • Secure user authentication and cart-based checkout. • Razorpay integration for live online payments (cards, UPI, netbanking, wallets). • Meta Pixel integration with all key e-commerce events for ad campaigns. • Real-time order status tracking for customers. • Full admin panel: orders, payments, users, broadcast emails. • Basic inventory system with stock tracking and Excel import/export. • Deployed live on the Diteup.com domain with SSL. 2. Order Tracking • Logged-in customers can track order status: Placed → Confirmed → Shipped → Delivered. • Status updates visible in real time from the user dashboard. • Automated status update emails to the customer. 2.2 Meta (Facebook) Pixel Integration Full Meta Pixel integration so the client can run Facebook and Instagram ads with complete conversion tracking and audience optimization. The following standard events will be fired: • PageView — every page visit. • ViewContent — when a user views the product. • AddToCart — when a user adds the product to cart. • InitiateCheckout — when a user starts the checkout process. • AddPaymentInfo — when a user enters payment details. • Purchase — fired on successful Razorpay payment confirmation (with order value and currency). • Lead / CompleteRegistration — when a user signs up. Pixel ID will be configurable from the admin panel — no code changes needed when switching pixels. Server-side fallback via Meta Conversions API is available as an optional add-on. 2.3 Admin Panel Order Management • List of all orders with search, filter (by status, date, customer), and pagination. • Detailed order view with customer, payment, and shipping info. • Update order status (Confirmed / Shipped / Delivered / Cancelled). • Export orders to Excel / CSV. • Import orders from Excel (bulk update use cases). Payment & Transactions • Complete payment history with Razorpay transaction IDs. • Filter by paid / failed / refunded / pending status. • Daily, weekly, and monthly revenue summary on dashboard. • Export transactions to Excel / CSV. User Management • List of all registered users with search and filter. • View individual user profile, orders, and lifetime value. • Enable / disable user accounts. • Export user list to Excel / CSV.
